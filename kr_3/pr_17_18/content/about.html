О приложении

Версия 1.0 — PWA с архитектурой App Shell

Заметки PWA

Простое и быстрое приложение для создания и управления заметками. Работает офлайн благодаря технологии Service Worker и кэшированию по стратегии App Shell.

Возможности

  • ✓ Работа без интернета (PWA / App Shell)
  • ✓ Сохранение заметок в localStorage
  • ✓ Кэширование статических ресурсов (Cache First)
  • ✓ Кэширование динамического контента (Network First)
  • ✓ Установка на рабочий стол (installable PWA)
  • ✓ Фолбек на главную страницу при отсутствии сети

Технологии

  • HTML5 / CSS3 / JavaScript
  • Service Worker + Cache API
  • App Shell Architecture
  • Fetch API для динамической загрузки контента
  • Manifest.json для PWA
